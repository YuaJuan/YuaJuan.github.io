给HCB看的python帮你玩跳一跳小白教程ヾ(≧▽≦*)o

大体步骤：

1.安装python运行环境及各种插件
2.使用adb驱动连接到手机调试
3.下载python程序并运行，OK！

是不是超级简单！

emmmmmm
其实也不是那么简单的。。。。。

1.安装python运行环境及各种插件
因为自动玩跳一跳的程序是用Python语言写的，所以我们需要安装python运行环境。
这里需要python3以及python的matplotlib和Pillow插件

1.1下载python3.6.4安装包

下载地址：
https://www.python.org/downloads/windows/

1.2安装python

反正就一直next,next，next……就好啦！
但是但是！一定要在这里勾选add paththon 3.6 to PATH ，不然就得自己配置环境变量！

检测python安装成功
打开命令行：win+r ，输入cmd后回车
在命令行输入python -V ，如下就是安装成功啦~

1.3安装需要的插件

windows+r 输入cmd ,打开cmd命令行。输入下面内容，就会自动安装matplotlib和pillow插件啦!

pip install matplotlib
pip install pillow

下载地址：https://github.com/wangshub/wechat_jump_game
