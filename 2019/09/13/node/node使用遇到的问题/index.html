创建软链接（注意这里必须要写绝对路径）

npm install -g truffle
但执行truffle后：

原因：
npm install xxx -g 时， 模块将被下载安装到【全局目录】中。
【全局目录】通过 npm config set prefix “目录路径” 来设置。
如：
npm config set prefix “/usr/“
那么再执行npm install -g truffle
就会被安装到/usr/bin/truffle
